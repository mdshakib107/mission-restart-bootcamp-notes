### 1. Concept Analysis
- এই Topic থেকে কোন কোন **core concept** শেখা দরকার
- প্রদত্ত Code গুলোতে কোন কোন ধারণা ব্যবহার হয়েছে
- প্রতিটি concept bullet point আকারে

### 2. Practice Question Design (Very Important)
প্রশ্নগুলো ৪টি লেভেলে ভাগ করবে:

#### A. Beginner Practice
- Concept বোঝার প্রশ্ন
- Small modification / prediction type
- “এই কোডে কী হবে?” টাইপ

#### B. Core Practice
- Code rewrite
- Logic apply করা
- Slight variation দিয়ে কাজ করানো

#### C. Intermediate Practice
- Real-world oriented task
- Multiple concept একসাথে ব্যবহার
- Edge case চিন্তা করতে বাধ্য করবে

#### D. Advanced / Thinking Practice
- “What if” scenarios
- Performance / behavior based
- Interview-style problems

### 3. Question Style Rules
- প্রতিটি প্রশ্ন এমন হবে যেনো:
  - লিখে উত্তর দিতে হয় **অথবা**
  - কোড লিখতে হয়
- MCQ এড়িয়ে চলবে
- প্রশ্ন difficulty ধাপে ধাপে বাড়বে
- চিন্তা করতে বাধ্য করবে

### 4. Practice Guidance (Without Giving Full Answer)
প্রতিটি প্রশ্নের নিচে আলাদা করে দেবে:
- **Hint / কী চিন্তা করতে হবে**
- **যে concept apply হবে**

❌ Full solution দেবে না  
✅ শুধুমাত্র learning-oriented guidance

### 5. Coverage Check
- Topic-এর কোনো গুরুত্বপূর্ণ অংশ বাদ যাবে না
- Code-এর সব গুরুত্বপূর্ণ behavior cover করতে হবে
