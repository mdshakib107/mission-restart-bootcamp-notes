JavaScript Memory: Stack vs Heap

Primitive value সরাসরি Stack-এ, Reference type-এর actual data Heap-এ

Stack fast, ordered memory frames
student = {
  name: "Alice",
  age: 22
}
Heap dynamic objects and arrays
let age = 25
student → 0xA13F
0xA13F
Step 1Primitive data যেমন number/string/boolean — Stack memory-তে রাখা হয়।
Step 2Object/Array/Function-এর actual value Heap memory-তে তৈরি হয়।
Step 3Stack-এর variable Heap object-এর address/reference ধরে রাখে।
