# 27. Even Number Problem

Problem:

```text
1 থেকে 100 পর্যন্ত even number এর sum বের করা
```

---

# 28. Even Number Logic

Even number check:

```js
number % 2 === 0;
```

---

# 29. Why?

কারণ even number কে 2 দিয়ে ভাগ করলে remainder হয়:

```text
0
```

---

# 30. Step-by-Step Logic

1. Loop 1 to 100
2. Check even
3. Add to sum

---

# 31. Full Solution

```js
let sum = 0;

for (let i = 1; i <= 100; i++) {
  if (i % 2 === 0) {
    sum += i;
  }
}

console.log(sum);
```

---

# 32. Output

```text
2550
```

---

# 33. Important Short-Hand

```js

```

Equivalent:

```js
sum = sum + i;
```

# 49. break Keyword

break ব্যবহার করা হয়:

```text
loop immediately stop করার জন্য
```

---

# 50. Example

```js
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    break;
  }

  console.log(i);
}
```

Output:

```text
1
2
```

---

# 51. Why 3 Printed হয়নি?

কারণ:

```js
break;
```

console.log এর আগেই execute হয়েছে।

---

# 52. continue Keyword

continue ব্যবহার করা হয়:

```text
current iteration skip করার জন্য
```

---

# 53. Example

```js
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue;
  }

  console.log(i);
}
```

---

# 54. Output

```text
1
2
4
5
```

---

# 55. Difference Between break and continue

| break              | continue                        |
| ------------------ | ------------------------------- |
| পুরো loop stop করে | শুধু current iteration skip করে |
| No more iterations | Next iteration continues        |

# 67. do-while Loop

Guarantees:

```text
At least one execution
```

---

# 68. do-while Syntax

```js

```

---

# 69. Important Difference

for / while:

```text
Condition আগে check হয়
```

do-while:

```text
Code আগে execute হয়
```

---

# 70. do-while Example

  number++;

# 72. Infinite Loop

---

# 73. Why Infinite Loop Happens?

কারণ:

- Condition সবসময় true
- Update missing
- Exit condition missing

---

# 74. Infinite for Loop

```js

```

---

# 75. Infinite while Loop

```js
while (true) {}
```

---

# 76. Infinite do-while Loop

```js
do {} while (true);
```

---

# 77. Problems with Infinite Loop

- CPU usage বেড়ে যায়
- Program hang করতে পারে
- Browser freeze হতে পারে

---

# 78. Important Advice

Beginner হিসেবে:

```text
Infinite loop avoid করো
```
