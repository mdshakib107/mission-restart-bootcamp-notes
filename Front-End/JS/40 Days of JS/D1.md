# 39. Problem with Script in Head

যখন script head এ থাকে:

- HTML পুরো parse হওয়ার আগে JS execute হয়
- নিচের element তখনও তৈরি হয়নি
- তাই JS element খুঁজে পায় না

Error:

```text
Cannot set properties of null
```

---

# 40. HTML Parsing Flow

Browser কাজ করে এভাবে:

1. HTML download
2. DOM build
3. Script download
4. Script execute

---

# 41. Fix: Put Script at End of Body

```html
<body>
  <div id="someId"></div>

  <script src="test.js"></script>
</body>
```

এখন HTML আগে parse হয়, পরে JS execute হয়।

---

# 42. async Attribute

```html
<script src="test.js" async></script>
```

Behavior:

- HTML parse চলতে থাকে
- JS parallel download হয়
- Download complete হলেই execute হয়

Problem:

- HTML parsing pause হতে পারে
- DOM পুরো ready নাও থাকতে পারে

---

# 43. async Use Case

External independent scripts এর জন্য।

Examples:

- Analytics script
- Chat widget
- WhatsApp floating button

---

# 44. defer Attribute

```html
<script src="test.js" defer></script>
```

Behavior:

- JS parallel download হয়
- HTML parsing continue হয়
- DOM পুরো ready হওয়ার পর JS execute হয়

---

# 45. Why defer is Best

Benefits:

- Performance better
- HTML parsing block হয় না
- DOM পুরো ready থাকে
- Error কম হয়

---

# 46. async vs defer

| Feature           | async                      | defer                       |
| ----------------- | -------------------------- | --------------------------- |
| Parallel download | Yes                        | Yes                         |
| Wait for DOM      | No                         | Yes                         |
| Execution timing  | Immediately after download | After HTML parsing complete |
| Best for          | Independent scripts        | Main app scripts            |
