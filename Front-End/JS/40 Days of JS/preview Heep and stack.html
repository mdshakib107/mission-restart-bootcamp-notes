Stack ও Heap Memory

দুই পাশে আলাদা example: Primitive বনাম Reference Type

Stack Example

Primitive values সরাসরি Stack-এ store হয়
let age = 25;
let name = "Alice";
let isStudent = true;
age = 25
name = "Alice"
isStudent = true

Heap Example

Object-এর actual data Heap-এ, Stack-এ থাকে reference
let student = {
  name: "Alice", age: 22
};
0xA13F → {
  name: "Alice",
  age: 22
}
student → 0xA13F
Stack reference
Stacknumber/string/boolean-এর মতো primitive value variable সহ Stack-এ রাখা হয়।
Heapobject/array/function Heap-এ থাকে; variable শুধু address বা reference ধরে রাখে।
