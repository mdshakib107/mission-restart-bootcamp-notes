# 115. Recursion

Definition:

```text
Function calling itself
```

---

# 116. Simple Example

```js
function foo() {
  foo();
}
```

---

# 117. Problem

Infinite recursion causes:

```text
Maximum call stack exceeded
```

---

# 118. Why Happens?

কারণ:

```text
Stack continuously grows
```

---

# 119. Base Condition

Recursion এ অবশ্যই দরকার:

```text
Exit condition
```

---

# 120. Recursion Example

```js
function fetchWater(count) {
  if (count === 0) {
    console.log("No more water");
    return;
  }

  console.log(count);

  fetchWater(count - 1);
}
```

---

# 121. Example Call

```js
fetchWater(5);
```

---

# 122. Output

```text
5
4
3
2
1
No more water
```

---

# 123. Important Recursion Concepts

| Concept        | Meaning                    |
| -------------- | -------------------------- |
| Recursive Call | Function calls itself      |
| Base Condition | Stops recursion            |
| Stack Growth   | Each call adds stack frame |
