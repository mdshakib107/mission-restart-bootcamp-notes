# 3. Operator

Operator হলো special symbol যা কোনো operation perform করে।

Examples:

```js
+
-
*
/
%
**
```

Example:

```js
x + y;
```

এখানে:

```text
+
```

হলো operator।

---

# 4. Operand

যার উপর operator কাজ করে তাকে operand বলে।

Example:

```js
x + y;
```

এখানে:

```text
x এবং y
```

দুইটিই operand।

---

# 5. Expression

যে code শেষ পর্যন্ত একটি value produce করে তাকে expression বলে।

Example:

```js
4 + 5;
```

Result:

```text
9
```

এটি একটি evaluating expression।

---

# 6. Assignment Expression

```js
let x = 10;
```

এখানে:

```text
=
```

হলো assignment operator।

এটি value assign করছে।

---

# 7. Evaluating Expression

```js
4 + 5;
```

এখানে:

- 4 → operand
- 5 → operand
- - → operator
- Result → 9

---

# 8. Combined Expression

```js

```

এখানে:

- Arithmetic evaluation হচ্ছে
- তারপর assignment হচ্ছে

# 20. Post Increment

```js

```

- পরে increment করে

# 36. Strict Equality

```js

```

---

# 37. Example

```js

```

- number
- string

different type।

---

# 38. Strict Equality Rules

1. Different type → false
2. null === null → true
3. undefined === undefined → true
4. NaN involved হলে → false

---

# 47. Truth Table

| A     | B     | Result |
| ----- | ----- | ------ |
| false | false | false  |
| true  | false | false  |
| true  | true  | true   |
| false | true  | false  |
