# 3. Variable কী?

Variable হলো programming এর এমন একটি storage/container যেখানে কোনো data বা information রাখা হয়।

বাস্তব উদাহরণ:

যদি তুমি বন্ধুর বাসায় যেতে চাও এবং বন্ধু ফোনে address বলে দেয়, তাহলে তুমি সেই address কোথাও লিখে রাখবে:

- কাগজে
- মোবাইল নোটে
- নোটপ্যাডে

মানে তুমি address store করলে।

Programming এও একইভাবে data store করতে হয়।

যেমন:

- User name
- Age
- Marks
- Address
- Employee salary

এই data গুলো store করার জন্য variable ব্যবহার করা হয়।

# 7. Variable Declaration Syntax

Modern JavaScript এ variable declare করার syntax:

```js
let storageName = value;
```

অথবা:

```js
const storageName = value;
```

অথবা পুরনো style:

```js
var storageName = value;
```

Example:

```js
let fruit = "Mango";
```

এখানে:

| Part      | Meaning                      |
| --------- | ---------------------------- |
| `let`     | Specifier / keyword          |
| `fruit`   | Variable name / storage name |
| `=`       | Assignment operator          |
| `"Mango"` | Value                        |

---

# 8. Specifier কী?

Variable declare করার সময় `let`, `const`, `var` ব্যবহার করা হয়।

এগুলোকে specifier বলা হয়েছে।

Specifier variable/storage কে control করে:

- কে access করতে পারবে
- কোথায় access করা যাবে
- value change করা যাবে কিনা
- variable আবার declare করা যাবে কিনা

JavaScript এ তিনটি main specifier:

```js
var
let
const
```

# 9. Assignment Operator

`=` হলো assignment operator।

এটি right side এর value কে left side এর variable এর মধ্যে assign করে।

Example:

```js
let fruit = "Mango";
```

এখানে:

```text
fruit = Mango
```

মানে `fruit` variable এর মধ্যে `"Mango"` রাখা হলো।

---

# 10. Value Reassignment

একই variable এ নতুন value assign করলে পুরনো value replace হয়ে যায়।

Example:

```js
let fruit = "Mango";
fruit = "Kiwi";
```

প্রথমে:

```text
fruit -> Mango
```

পরে:

```text
fruit -> Kiwi
```

মানে Mango বের হয়ে Kiwi ঢুকে গেল।

# 12. Variable to Variable Assignment

Example:

```js
let fruit = "Mango";
let vegetable = "Carrot";

fruit = vegetable;
```

এখানে `vegetable` এর value `fruit` এর মধ্যে assign হচ্ছে।

Result:

```text
fruit -> Carrot
vegetable -> Carrot
```

Important:

- Left side variable change হয়
- Right side variable change হয় না

---

# 13. Pass by Value

Primitive data type এর ক্ষেত্রে assignment হয় **pass by value**।

Example:

```js
let fruit = "Mango";
let vegetable = "Carrot";

fruit = vegetable;
```

এখানে:

- `vegetable` এর value `"Carrot"` copy হয়ে `fruit` এ যায়
- `vegetable` নিজে unchanged থাকে

Result:

```js
console.log(fruit); // Carrot
console.log(vegetable); // Carrot
```

Pass by value শুধু primitive type এর ক্ষেত্রে ঘটে।

Numeric value।


---

## 15.3 Boolean

Boolean value শুধু দুই রকম হতে পারে:

```js

```

Example:

```js

```

---

## 15.4 Undefined

যখন variable declare করা হয় কিন্তু value assign করা হয় না, তখন তার value হয় `undefined`।

Example:

```js
let salary;

console.log(salary);
```

Output:

```text
undefined
```

মানে variable আছে, কিন্তু value define করা হয়নি।

---

## 15.5 Null

`null` মানে intentional empty value।

Example:

```js
let salary = null;
```

এখানে salary variable declare করা হয়েছে এবং value হিসেবে `null` assign করা হয়েছে।

Difference:

| Undefined                | Null                                   |
| ------------------------ | -------------------------------------- |
| Value assign করা হয়নি    | Empty value ইচ্ছা করে assign করা হয়েছে |
| JavaScript নিজে দেয়      | Developer নিজে দেয়                     |
| Declared but not defined | Defined as empty                       |

Example:

```js
let name;
let salary = null;
```

```js
console.log(name); // undefined
console.log(salary); // null
```

# 20. Variable Naming Rules

JavaScript এ variable name দেওয়ার কিছু grammar/rules আছে।

Programming language এর grammar না মানলে JavaScript error দেয়।

---

## 20.1 Allowed Characters

Variable name এ থাকতে পারে:

- Letters: `a-z`, `A-Z`
- Digits: `0-9`
- Underscore: `_`
- Dollar sign: `$`

Example:

```js
let name = "Rahim";
let user1 = "Karim";
let _age = 20;
let $price = 100;
```

---

## 20.2 Number দিয়ে শুরু করা যাবে না

Invalid:

```js
let 2name = "Rahim";
```

কারণ variable name digit দিয়ে শুরু হতে পারে না।

Valid:

```js
let name2 = "Rahim";
```

---

## 20.3 Hyphen ব্যবহার করা যাবে না

Invalid:

```js
let react-play = "course";
```

কারণ `-` allowed special character না।

Valid:

```js
let reactPlay = "course";
```

---

## 20.4 Reserved Keywords ব্যবহার করা যাবে না

JavaScript এর reserved keyword variable name হিসেবে ব্যবহার করা যাবে না।

Invalid:

```js
let for = 10;
let if = true;
let else = "no";
```

কারণ `for`, `if`, `else` JavaScript language এর reserved keyword।

# 26. var, let, const

JavaScript এ variable declare করার তিনটি keyword:

```js
var
let
const
```

এই লেকচারে scope detail এ আলোচনা করা হয়নি।

Scope পরে শেখানো হবে।

এখানে basic difference শেখানো হয়েছে:

| Keyword | Redeclare | Reassign |
| ------- | --------- | -------- |
| `var`   | Yes       | Yes      |
| `let`   | No        | Yes      |
| `const` | No        | No       |

---

# 27. var

`var` দিয়ে variable redeclare করা যায়।

Example:

```js
var address = "Bangalore";
console.log(address);

var address = "USA";
console.log(address);
```

Output:

```text
Bangalore
USA
```

Problem:

- একই variable বারবার declare করলে confusion তৈরি হয়
- বড় codebase এ bug হতে পারে
- modern JavaScript এ `var` avoid করা ভালো

---

# 28. let

`let` দিয়ে variable redeclare করা যায় না, কিন্তু reassign করা যায়।

Valid:

```js
let address = "Bangalore";
console.log(address);

address = "USA";
console.log(address);
```

Output:

```text
Bangalore
USA
```

Invalid:

```js
let address = "Bangalore";
let address = "USA";
```

Error:

```text
Identifier 'address' has already been declared
```

---

# 29. const

`const` দিয়ে variable declare করলে value reassign করা যায় না।

Valid:

```js
const address = "Bangalore";
console.log(address);
```

Invalid:

```js
const address = "Bangalore";
address = "USA";
```

Error:

```text
TypeError: Assignment to constant variable
```

`const` মানে constant value।

# 30. Declare vs Reassign vs Redeclare

## Declare

Variable প্রথমবার তৈরি করা।

```js
let age;
```

## Assign

Variable এ value রাখা।

```js
age = 20;
```

## Declare + Assign

একই সাথে variable create এবং value assign করা।

```js
let age = 20;
```

## Reassign

আগে declared variable এ নতুন value দেওয়া।

```js
age = 25;
```

## Redeclare

একই variable name আবার keyword দিয়ে declare করা।

```js
let age = 20;
let age = 25; // invalid with let
```

# 35. Stack Memory

Primitive values stack memory তে store হয়।

```js

```

# 36. Heap Memory

Non-primitive/reference values heap memory তে store হয়।

Example:

```js
let student = {
  name: "Alice",
  age: 22,

```

Object এর actual value heap memory তে থাকে।

Variable reference/address ধরে রাখে।

| Heap   | Non-Primitive values | object, array, function |

# 41. Parsing

---

# 42. Abstract Syntax Tree বা AST

AST হলো code এর tree representation।

Example code:

```js
let x = 10;
```

এই code থেকে AST তৈরি হয়।

AST-তে থাকে:

- Variable declaration
- Variable kind: `let`
- Identifier: `x`
- Value: `10`
- Start position
- End position

# 48. Main Learning from var, let, const

## var

```js
var address = "Bangalore";
var address = "USA";
```

Allowed, কিন্তু avoid করা ভালো।

## let

```js
let address = "Bangalore";
address = "USA";
```

Allowed।

কিন্তু:

```js
let address = "Bangalore";
let address = "USA";
```

Not allowed।

## const

```js
const address = "Bangalore";
address = "USA";
```

Not allowed।

- `var` avoid করো
